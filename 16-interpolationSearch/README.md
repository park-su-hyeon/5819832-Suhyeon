 Interpolation Search가 Binary Search보다 적게 비교하는 이유
보간 탐색이 타멧의 위치를 추정하여 탐색하기 때문에 더 효율적으로 비교할 수 있다.
이진탐색은 타멧의 위치에 관계없이 항상 동일하게 진행되고, 보감 탐색은 타켓의 범주를 정확하게 추정할 수 있다면 훨씬 더 적은 비교로 찾을 수 있다.
